# Embedding Universe

An interactive semantic network visualization tool that is cool to look at.

## What it does

- Add words and see their connections based on AI embeddings
- Visualize semantic relationships in real-time
- Explore how similar words cluster together
- All processing happens in your browser - no server needed

## How to run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and start adding words.

## Features

- **Browser-based AI** - Uses Transformers.js, no external API calls all in the browser
- **Interactive canvas** - Drag to pan, scroll to zoom
- **Live connections** - Words connect based on semantic similarity
- **Educational** - Learn how AI models understand language relationships
